# Playbook Selection Guide — Full Decision Matrix

## The Scoring System

Rate each playbook 1-10 for the specific product. The highest score wins.

---

## Factor 1: Price Point

| Price Range | Dark Premium | Elegant Authority | Clean Light | White DR | Dark Funnel |
|-------------|:-----------:|:-----------------:|:-----------:|:--------:|:-----------:|
| Free / Lead magnet | 3 | 2 | 6 | **10** | 5 |
| Under ₪100 | 4 | 3 | 7 | **9** | 6 |
| ₪100–₪500 | 7 | 5 | **8** | 6 | 7 |
| ₪500–₪2,000 | **8** | 7 | 7 | 4 | **8** |
| ₪2,000–₪5,000 | **9** | **9** | 5 | 2 | 6 |
| ₪5,000+ | 8 | **10** | 3 | 1 | 4 |

**Why:** Low-ticket = reduce friction, let copy sell → White DR. High-ticket = build trust/authority → Elegant Authority. Mid-range = energy/action → Clean Light or Dark Premium.

## How to Calculate

1. Score each factor (1-10) using the tables above
2. Apply weights:
   - Price + Audience: multiply by **1.5** (most important)
   - Product type + Vibe: multiply by **1.0**
   - Copy tone: multiply by **0.8**
3. Sum totals per playbook
4. Highest score = primary recommendation
5. Second highest = alternative

### Quick Mental Math

In practice, you usually don't need the full matrix. Ask yourself:

1. **Is it low-ticket (under ₪100)?** → White DR or Clean Light
2. **Is it high-ticket consulting/coaching?** → Elegant Authority
3. **Is it tech/AI/modern?** → Dark Premium
4. **Is it affiliate/launch/countdown?** → Dark Funnel
5. **Is it a course for a wide audience?** → Clean Light

If none is obvious, run the full matrix.

## Brand Color Adaptation

When the user has existing brand colors, adapt the playbook:

### Step 1: Classify the Brand Color

| Color Temperature | Examples | Natural Playbook |
|------------------|----------|-----------------|
| Warm dark (navy, dark blue, charcoal) | #1D2B46, #2C3E50, #333 | Elegant Authority |
| Cold dark (black, dark gray) | #0a0a0a, #1a1a1a | Dark Premium |
| Warm accent (gold, orange, amber) | #d4af37, #f59e0b, #e67e22 | Dark Premium or Elegant Authority |
| Cool accent (blue, teal, purple) | #3498db, #1abc9c, #9b59b6 | Clean Light or Dark Premium |
| Green | #27ae60, #73B96B | Clean Light |
| Red/Pink | #e74c3c, #E4C0C1 | Dark Funnel (red) or White DR (pink) |
| Neutral (gray, beige, cream) | #f2f2f2, #f5f0e8 | Clean Light or White DR |

### Step 2: Slot Into Playbook

```
Brand color → Accent/CTA position
Playbook's original bg → Keep as-is
Playbook's original text → Keep as-is (contrast matters more)
```

### Step 3: Verify Contrast

- CTA button text must have 4.5:1 contrast ratio minimum
- Accent on dark bg: use lighter shade
- Accent on light bg: use darker shade
- When in doubt: test at https://webaim.org/resources/contrastchecker/

## Common Hybrid Combinations

Sometimes the best answer is mixing 2 playbooks:

| Hybrid | When to Use |
|--------|-------------|
| Dark Premium base + Elegant Authority cards | Tech product with trust-heavy copy |
| Clean Light base + Dark Premium hero | Approachable product with dramatic opening |
| Elegant Authority base + custom brand colors | Personal brand with specific visual identity |
| White DR base + Clean Light CTA style | Long-form sales letter for a course |

**Rule:** Pick ONE base playbook for structure/layout, borrow only colors or specific components from the second. Never mix layouts.

## Red Flags — When to Override the Matrix

| Signal | Override to |
|--------|------------|
| User says "אני רוצה משהו כהה" | Dark Premium or Dark Funnel |
| User says "משהו נקי ואלגנטי" | Elegant Authority or Clean Light |
| User says "משהו יוקרתי ומקצועי" | Elegant Authority |
| User has a strong brand with specific colors | Hybrid with their colors |
| The copy is 3000+ words | White DR (only style that handles this) |
| There's a countdown/timer element | Dark Funnel |
| Target audience is 55+ | Clean Light or White DR (readable, simple) |
