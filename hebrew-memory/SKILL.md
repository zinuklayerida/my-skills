# זיכרון עברית — Hebrew Skills Memory

כשהמשתמש מריץ את הסקיל הזה, שמור את המיפוי הבא בזיכרון של Claude Code (CLAUDE.md או memory) כדי שקלוד ידע לקשר מילים בעברית לסקילים הנכונים.

**הוראות:** העתק את המיפוי הבא לקובץ `CLAUDE.md` בתיקיית הפרויקט, או שמור בזיכרון.
